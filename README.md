**Description**

Simple Angular 1.5 exercise for basic CRUD operations.
Tested best practices including modules, submodules and component-like approach.

**Technologies used**
- AngularJS 1.5
- ASP.NET Core 1.0 
- Entity Framework Core 1.0 (with Microsoft's SQLite provider)
- Bulma

**Worklog**
